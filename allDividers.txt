Задача: найти все положительные делители заданного числа

Алгоритм:
1. Считать число n
2. Пройти по всем числам от 1 до n

  2.1. Проверить, является ли текущее число делителем числа n
  2.2. Если да, то вывести текущее число

read n;
i := 1;
M1: if i <= n {
	if n mod i = 0 {
		write i;
	}
	inc (i);
	goto M1;
}
